## ğŸ“ Customization

### Update Personal Information

1. **Contact Details**: Edit `src/components/Hero.jsx` and `src/components/Contact.jsx`
2. **Social Links**: Update LinkedIn and GitHub URLs in:
   - `src/components/Hero.jsx`
   - `src/components/Contact.jsx`
3. **Experience**: Modify the `experiences` array in `src/components/Experience.jsx`
4. **Projects**: Update the `projects` array in `src/components/Projects.jsx`
5. **Skills**: Adjust the `skillCategories` array in `src/components/Skills.jsx`
6. **Education**: Edit the `education` and `certifications` arrays in `src/components/Education.jsx`

### Styling

- Main styles: `src/index.css`
- Component-specific styles: Each component has its own CSS file in `src/components/`
- Color scheme: Modify CSS variables in `src/index.css` (`:root` section)
